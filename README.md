SQL analysis of a digital music store uncovering sales trends, customer insights, and agent performance through 26 business driven queries
# 🎵 Chinook Music Store:  SQL Analysis

## 📌 Overview
This project is a business-driven SQL analysis of the **Chinook database**,
a sample database simulating a digital music store (similar to iTunes).

The goal is to answer concrete business questions through **26 SQL queries**
organized by theme, using **Microsoft SQL Server**.

---
## 🛠️ Tools & Technologies
- **Language** : SQL
- **Database** : Chinook Database
- **Tool** : Microsoft SQL Server Management Studio (SSMS)

---
## 📊 Project Structure

### 🧑‍🤝‍🧑 Theme 1: Customer Analysis
> Understanding the customer base and their geographic distribution

- International customers outside the USA
- Brazilian customers and their contact details
- Invoices from Brazilian customers
- Identification of sales support agents

### 💰 Theme 2: Sales Analysis
> Analyzing commercial performance and transactions

- Active markets overview
- Invoices per sales agent
- Full transaction report
- Sales analysis by year
- Line items per invoice

### 🎵 Theme 3: Tracks & Artists
> Exploring the music catalog and purchasing trends
- Track details per invoice
- Tracks and artists per invoice
- Invoice distribution by country
- Playlist popularity
- Full track catalog

### 📊 Theme 4: Reporting & Aggregations
> Producing automated reports for management

- Global invoice overview
- Total sales per agent
- Total sales by country

### 🏆 Theme 5: Agent Performance
> Evaluating and comparing commercial performance
- Best sales agent per year (2021, 2022)
- Best sales agent overall
- Top 5 most purchased tracks
- Top 3 best-selling artists
- Most purchased media type

## 💡 SQL Skills Demonstrated
| Skill | Examples |
|---|---|
